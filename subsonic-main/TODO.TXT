CrashPlan
---------
To make it work on subsonic.org:
- Put jre in /usr/local/crashplan/jre (copy from forum.subsonic.org)
- Install libffi for Centos
- Extract libjnidispatch.so from jna-3.2.5 and put it in /usr/local/crashplan
